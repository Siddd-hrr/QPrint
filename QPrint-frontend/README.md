# QPrint Frontend

Vite + React 18 + Tailwind + Router + Zustand + Axios scaffold per Phase 1 spec.

## Scripts
- `npm run dev` — start dev server on 5173
- `npm run build` — production build
- `npm run preview` — preview build

## Env
Copy `.env.example` to `.env` and set `VITE_API_BASE_URL` (default points to gateway at 8080).
